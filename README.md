# 🤖 NIKK AI — Personal Voice Assistant

NIKK AI is a browser-based personal voice assistant built with pure HTML, CSS, and JavaScript. It listens to your voice commands, responds with speech, fetches live weather, opens websites, and much more — all inside your browser with zero dependencies.

---

## ✨ Features

- 🎤 **Voice Recognition** — Start/Stop listening with one click
- 🔊 **Text-to-Speech** — NIKK responds out loud to every command
- 🌤️ **Live Weather** — Real-time weather data for any city
- 🤖 **Animated Avatar** — Blinking eyes, talking mouth, spinning rings
- 💬 **Chat History** — Full conversation log with bubble UI
- 🌙 **Dark / Light Mode** — Toggle anytime, preference saved
- 🌌 **Glassmorphism UI** — Deep space background with twinkling stars and glowing orbs
- 📱 **Responsive** — Works on desktop and mobile

---

## 🗣️ Voice Commands

| Command | Action |
|---|---|
| `hi nikk` / `hello nikk` | Greeting |
| `open youtube` | Opens YouTube |
| `open google` | Opens Google |
| `open instagram` | Opens Instagram |
| `open github` | Opens GitHub |
| `my github` | Opens your GitHub profile |
| `my instagram` | Opens your Instagram profile |
| `search [query]` | Google search |
| `search [query] on youtube` | YouTube search |
| `weather in [city]` | Weather for any city |
| `what time` | Current time |
| `today's date` | Current date |
| `my name` | Tells your saved name |
| `stop listening` | Stops voice recognition |

---

## 🛠️ Tech Stack

- **HTML5** — Structure
- **CSS3** — Glassmorphism, animations, dark/light mode
- **Vanilla JavaScript** — All logic, no frameworks
- **Web Speech API** — Voice recognition + Text-to-speech
- **OpenWeather API** — Live weather data

## 📂 Project Structure

```
nikk-ai/

└── app.js       # Core logic & voice commands
```

---

## ⚙️ How to Run

1. Download all 3 files into one folder
2. Open `index.html` in your browser (Chrome recommended)
3. Fill in your profile on first launch
4. Click **Start** and speak your command!

> ⚠️ Voice recognition requires microphone permission and works best in **Google Chrome**.
