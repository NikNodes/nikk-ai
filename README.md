# 🤖 Nikk AI - Voice Assistant Web App

Nikk AI is a browser-based voice assistant that can listen to user commands, respond with voice, and perform tasks like fetching weather information.

## 🚀 Features

- 🎤 Voice recognition (Start/Stop listening)
- 🔊 Text-to-speech response
- 🌦️ Weather information using API
- 💬 Typing animation effect
- 🎨 Interactive UI with visual feedback
- ⚡ Real-time response system

## 🛠️ Tech Stack

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)  
- Web Speech API (Speech Recognition + Speech Synthesis)  
- OpenWeather API  

## 📂 Project Structure
├── index.html # Main UI structure
├── style.css # Styling and animations
├── app.js # Core logic and functionality
